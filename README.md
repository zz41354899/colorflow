# ✨ Vibe Name Generator

產生你的 vibe 名字吧！這是一個簡單而有趣的動畫互動頁，使用者只需點擊按鈕，即可隨機獲得風格化的 vibe 名，配合漸變背景與文字動畫，讓每一次點擊都像儀式感啟動。


## 🔧 使用方式

1. 下載資料夾（`vibe-name-generator`）
2. 直接打開 `index.html` 檔案即可使用
3. 無需安裝任何函式庫，已經整合 CDN

## 🧪 技術架構

- HTML + Tailwind CSS（CDN）
- GSAP 動畫（CDN）
- JavaScript 文字隨機 + 動畫切換邏輯
